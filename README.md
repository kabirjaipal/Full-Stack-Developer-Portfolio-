## **Installation | How to use the Bot**

**1.** Install [NodeJS](https://nodejs.org/en/)

**2.** Download this repo and unzip it | or git clone it

**3.** Install NPM Modules **`npm install`**

**4.** start the website with **`npm run dev`**
